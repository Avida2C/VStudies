PHP - PHP Hypertext Preprocessor

A scripting / server side language which runs on the server side which means, it runs on a server NOT on a local computer. Unlike HTML, CSS and Javascript which run on the client side, which means on the browser.

PHP is a loose language, one of the easiest programming languages. Although the code may be badly coded, it can still function. It may cause the lack of security due to inexperience.

Requirements

  • HTML
  • JavaScript (recommended)
  • Local Server (XAMPP)
